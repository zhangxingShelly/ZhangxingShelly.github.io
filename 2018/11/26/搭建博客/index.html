搭建个人博客的过程笔记（基于Hexo和Github）

  之前周围有几个朋友给我看了他们搭建的个人博客，也建议我有机会可以自己搭建一个。看着感觉很好玩，想着有时间一定要自己试试。现在总算是开始了这个尝试。我是一个容易忘事的人，简单记录一下过程。之后万一有需要，也方便复现，哈哈。

一、电脑环境搭建

  1. 安装Node.js：为Hexo提供js脚本的运行环境。
  2. 安装Yarn：Yarn是JavaScript的包管理工具。
  3. 安装Git。
  4. 安装Hexo：通过Yarn来全局安装Hexo

  具体安装步骤参考博客。

二、使用Hexo创建静态博客

  1. 创建站点目录：选择一个目录作为Hexo站点目录，之后主题配置文件、创建博文的md文件都存储在本目录下。我选择在E盘新建一个MyBlog目录，作为Hexo站点目录。
  2. 初始化Hexo：运行cmd,cd 到站点目录,输入命令：hexo init。
  3. 更换博客主题：初始化Hexo之后，在你创建的站点目录下会出现名为themes的文件夹，文件下有名为landscape的文件夹，内含Hexo自带的博客主题风格配置文件。我们可选择自己喜欢的博客主题风格进行更换。
    主题汇总：可在Github上查看各种主题，进行选择。
    本博客使用的主题为Pacman提供。
    在cmd中使用如下命令将新的主题配置文件复制到themes文件夹下

    ‘ git clone https://github.xom/A-limon/pacman.git themes/pacman ‘

    然后打开站点目录下的配置文件_config.yml，打开它(可使用Notepad++打开)。在文件中找到themes:landscape，将其改为themes:pacman

    执行hexo clean 和 hexo g 以及 hexo s 生成静态博客，可以在本地浏览器中输入url:localhost:4000查看生成效果。

三、部署到Github

  1. 注册Github，获取Github Pages，步骤可自行百度
  2. 安装插件：在cmd中cd到站点目录，然后使用yarn安装插件

    Yarn add hexo-deployer-git
    如果使用的是包管理器npm进行安装，则使用的命令是：
    npm install hexo-deployer-git –save
    然后配置 Hexo根目录的 _config.yml, 在文件中增加如下代码:
    ‘’’
    deploy:
    type: git
    repo: https:// github.com/your_username/your_username.github.io.git
    branch: master
    ‘’’
    如果你是第一次配置 Github 远程仓库，你还须将你电脑的SSH Key 配置到 Github 上:

  3. 打开 git bash ，输入 ssh-keygen -t rsa -C “你的github注册邮箱”，连续三个回车，生成密钥，最后得到了两个文件：id_rsa和id_rsa.pub（默认存储路径是：C:\Users\Administrator.ssh）。

  4. 登录Github，点击头像下的settings，点击 SSH and GPG keys ，点击 New SSH key，将id_rsa.pub文件里的内容复制上去。title可以随意填或不填。点击add key按钮，在弹出的对话框输入Github账号的密码即可。

  5. 部署到Github：执行hexo g 然后执行hexo deploy，在弹出的对话框中输入github账号和密码即可。部署成功后在浏览器输入网址：your_username.github.io就可以查看你创建的博客啦。
